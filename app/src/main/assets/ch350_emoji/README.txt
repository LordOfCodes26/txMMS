Bundled CH350 emoticon resource packs (optional)

Drop one or more CH350-format *.obb files here (same files the 350 SMS app uses with libresutils).
They are copied to app storage on first emoji open. The CH350 Git project usually does NOT include
these binaries; obtain them from your CH350 product build or device image.

Native library: add libresutils.so per ABI under app/src/main/jniLibs/ (see jniLibs/README.txt).
